## Setup Instructions

### متطلبات النظام
- لغة برمجة بايثون
- قاعدة بيانات SQL

### System Requirements
- Python programming language
- SQL database

### خطوات التثبيت
1. استنساخ المستودع
   ```bash
   git clone https://github.com/Drbiowner/coffee-shop-system.git
   ```
2. تثبيت الحزم المطلوبة
   ```bash
   pip install -r requirements.txt
   ```

### Installation Steps
1. Clone the repository
   ```bash
   git clone https://github.com/Drbiowner/coffee-shop-system.git
   ```
2. Install the required packages
   ```bash
   pip install -r requirements.txt
   ```
